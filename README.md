## PolyFactorGF — факторизация многочленов в GF(p)

Курсовая работа (ГУАП, 2 курс): консольное TUI‑приложение на C++ для факторизации многочленов над конечным полем GF(p), где p — простое число.[^2]

Программа умеет:

- разлагать многочлен на неприводимые множители над GF(p),
- корректно обрабатывать кратные корни,
- запускать пакетные тесты из файлов `tests/tests.txt`, `tests/stresstest.txt`, `tests/stress2.txt`.[^1]

***

## Основная логика

Вся математика собрана в классе `PolyFactorService`:

```cpp
using Poly = std::vector<long long>;

class PolyFactorService {
public:
    std::string factorize(long long p, const Poly& poly);
    ...
};
```


### Что делает `PolyFactorService`

1. **Арифметика в GF(p)**
Методы `mod`, `modpow`, `modinv` реализуют:
    - безопасное приведение по модулю p,
    - быстрое возведение в степень,
    - обратный элемент по малой теореме Ферма (для простого p).
2. **Многочленная арифметика в GF(p)[x]**

Методы:
    - `polyAdd`, `polySub`, `polyMul` — сложение, вычитание и умножение многочленов;
    - `polyDiv`, `polyMod` — деление с остатком (частное и остаток);
    - `polyGCD` — НОД многочленов;
    - `polyDeriv` — производная многочлена;
    - `normalize`, `degree` — нормализация (унитарный многочлен, удаление ведущих нулей) и вычисление степени.
3. **Square‑free разложение**

`squareFreeDecompose` выделяет бесквадратную часть и кратности корней:
    - если производная равна 0, используется понижение степени $f(x) = g(x^p)$;
    - иначе строится стандартное square‑free разложение $f = \prod f_i^{m_i}$.

Это нужно, чтобы корректно находить кратные корни (например, $x^2$, $x^3$, $(x+2)^2$).
4. **Алгоритм Берлекэмпа**

Метод `berlekamp` реализует факторизацию бесквадратного многочлена над GF(p):
    - строится матрица $B$, строки которой — остатки $x^{ip} \mod f(x)$;
    - из неё формируется матрица $Q = B^T - I$;
    - методом Гаусса находится ядро $Q$, то есть базис многочленов $h(x)$, удовлетворяющих $h(x)^p ≡ h(x) \mod f(x)$;[^2]
    - методом `splitFactors` текущий многочлен рекурсивно расщепляется через НОД $\gcd(f(x), h(x)-c)$ для всех $c \in GF(p)$.
5. **Полная факторизация**

`fullFactorize`:
    - сначала делает square‑free разложение,
    - затем к каждой бесквадратной части применяет `berlekamp`,
    - собирает список пар *(множитель, кратность)*.
6. **Формирование текстового вывода**
    - `polyToString` переводит многочлен в строку вида `x^4 + 3x^2 + 1`;
    - `factorize(p, poly)`:
        - печатает исходный многочлен и поле GF(p);
        - если степень 0 — пишет «Константа, факторизация невозможна.»;
        - если получен один множитель той же степени — пишет «Многочлен неприводим над GF(p).»;
        - иначе выводит:
            - строку `Разложение на множители: f(x) = ...`,
            - список множителей `f1(x) = ...`, с указанием кратности, если она больше 1.

## TUI-интерфейс (main.cpp)

Консольный интерфейс предоставляет простое текстовое меню:

```text
=== PolyFactorGF — TUI ===
1) Факторизовать многочлен
2) Загрузить из файла (каждая строка: p coeffs)
3) Выход
```

1. **Режим 1 — ручной ввод**
    - Пользователь вводит простое $p$.
    - Затем вводит коэффициенты многочлена через пробел, от младшего к старшему (например `6 0 1` для $x^2+6$).
    - Вызвается `PolyFactorService::factorize(p, poly)` и выводится полный результат.
2. **Режим 2 — пакетная загрузка из файла**
    - Пользователь вводит путь к файлу (например, `tests/tests.txt`).
    - Каждая непустая строка файла должна иметь формат:

```text
p c0 c1 c2 ... cn
```

где p — характеристика поля, а `c0..cn` — коэффициенты многочлена от младшего члена к старшему.
    - Для каждой строки:
        - печатается заголовок `--- p=... coeffs=... ---`,
        - выполняется факторизация и вывод результата.
3. **Режим 3 — выход**
    - Завершает работу программы.

Для очистки экрана используется ANSI-последовательность `\x1B[2J\x1B[H`, поэтому наиболее красиво это выглядит в нормальном терминале (Linux/macOS).

## Формат файлов тестов

Каждая строка в `tests.txt`, `stresstest.txt`, `stress2.txt`:

```text
p c0 c1 c2 ... cn
```

где:

- `p` — характеристика поля (простое число),
- `c0` — свободный член (коэффициент при $x^0$),
- `c1` — коэффициент при $x^1$,
- ...
- `cn` — коэффициент при $x^n$.

Например:

```text
7 6 0 1       # x^2+6 над GF(7)
5 0 0 1       # x^2 над GF(5)
2 1 1 1       # x^2+x+1 над GF(2)
```

## Теория

Факторизация многочленов над конечным полем GF(p) — задача поиска представления

$$
f(x) = \prod_{i} f_i(x)^{m_i},
$$

где $f_i(x)$ — неприводимые многочлены над GF(p), а $m_i$ — их кратности.[^2]

В проекте используется алгоритм Берлекэмпа для факторизации бесквадратной части и square‑free decomposition для обработки кратных корней.[^2]
